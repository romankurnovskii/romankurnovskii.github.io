<mark>[1772A - A+B? (implementation, 800)](https://codeforces.com/contest/1772/problem/A)</mark>

## Solution

```python
def solve():
    a, b = input().split('+')
    res = int(a) + int(b)
    print(res)

for _ in range(int(input())):
    solve()
```

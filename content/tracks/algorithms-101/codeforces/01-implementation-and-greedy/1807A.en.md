<mark>[1807A - Plus or Minus (implementation, 800)](https://codeforces.com/contest/1807/problem/A)</mark>

## Solution

```python
t = int(input())

for _ in range(t):
    a, b, c = map(int, input().split())
    print('+' if a + b == c else '-')
```

[LeetCode problem]()

Given the 'head' of a linked list, remove the 'nth' node from the end of the list and return its head.


**Example 1:**

![LeetCode 19. Remove Nth Node From End of List](https://assets.leetcode.com/uploads/2020/10/03/remove_ex1.jpg)

    Input: head = [1,2,3,4,5], n = 2
    Output: [1,2,3,5]


**Example 2:**

    Input: head = [1], n = 1
    Output: []


**Idea:**

- Two pointers.
- Second pointer starts from `nth` position.
- Run while second pointer exist.

First version:

```python
# Definition for singly-linked list.
# class ListNode:
#     def __init__(self, val=0, next=None):
#         self.val = val
#         self.next = next
class Solution:
    def removeNthFromEnd(self, head, n: int):
        p1 = head
        p2 = head

        for _ in range(n):
            p1 = p1.next

        if not p1:
            return head.next # in case: head=[1], n=1 -> return []

        while p1.next:
            p1 = p1.next
            p2 = p2.next

        p2.next = p2.next.next

        return head
```

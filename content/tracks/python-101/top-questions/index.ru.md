### 22. –ß—Ç–æ —Ç–∞–∫–æ–µ –¥–µ–∫–æ—Ä–∞—Ç–æ—Ä—ã –≤ Python?

–î–µ–∫–æ—Ä–∞—Ç–æ—Ä—ã –≤ Python - —ç—Ç–æ, –ø–æ —Å—É—Ç–∏, —Ñ—É–Ω–∫—Ü–∏–∏, –∫–æ—Ç–æ—Ä—ã–µ –¥–æ–±–∞–≤–ª—è—é—Ç —Ñ—É–Ω–∫—Ü–∏–æ–Ω–∞–ª—å–Ω–æ—Å—Ç—å –∫ —Å—É—â–µ—Å—Ç–≤—É—é—â–µ–π —Ñ—É–Ω–∫—Ü–∏–∏ –≤ Python –±–µ–∑ –∏–∑–º–µ–Ω–µ–Ω–∏—è —Å—Ç—Ä—É–∫—Ç—É—Ä—ã —Å–∞–º–æ–π —Ñ—É–Ω–∫—Ü–∏–∏. –í Python –æ–Ω–∏ –æ–±–æ–∑–Ω–∞—á–∞—é—Ç—Å—è @decorator_name –∏ –≤—ã–∑—ã–≤–∞—é—Ç—Å—è –ø–æ –ø—Ä–∏–Ω—Ü–∏–ø—É "—Å–Ω–∏–∑—É –≤–≤–µ—Ä—Ö". –ù–∞–ø—Ä–∏–º–µ—Ä:

```python
# decorator function to convert to lowercase
def lowercase_decorator(function):
   def wrapper():
       func = function()
       string_lowercase = func.lower()

   return wrapper

       func = function()
       string_split = func.split()

    return 'Hello World'

```

–ü—Ä–µ–ª–µ—Å—Ç—å –¥–µ–∫–æ—Ä–∞—Ç–æ—Ä–æ–≤ –∑–∞–∫–ª—é—á–∞–µ—Ç—Å—è –≤ —Ç–æ–º, —á—Ç–æ –ø–æ–º–∏–º–æ –¥–æ–±–∞–≤–ª–µ–Ω–∏—è —Ñ—É–Ω–∫—Ü–∏–æ–Ω–∞–ª—å–Ω–æ—Å—Ç–∏ –∫ –≤—ã—Ö–æ–¥—É –º–µ—Ç–æ–¥–∞, –æ–Ω–∏ –º–æ–≥—É—Ç –¥–∞–∂–µ –ø—Ä–∏–Ω–∏–º–∞—Ç—å –∞—Ä–≥—É–º–µ–Ω—Ç—ã –¥–ª—è —Ñ—É–Ω–∫—Ü–∏–π –∏ –¥–æ–ø–æ–ª–Ω–∏—Ç–µ–ª—å–Ω–æ –º–æ–¥–∏—Ñ–∏—Ü–∏—Ä–æ–≤–∞—Ç—å —ç—Ç–∏ –∞—Ä–≥—É–º–µ–Ω—Ç—ã –ø–µ—Ä–µ–¥ –ø–µ—Ä–µ–¥–∞—á–µ–π –≤ —Å–∞–º—É —Ñ—É–Ω–∫—Ü–∏—é. –í–Ω—É—Ç—Ä–µ–Ω–Ω—è—è –≤–ª–æ–∂–µ–Ω–Ω–∞—è —Ñ—É–Ω–∫—Ü–∏—è, —Ç–æ –µ—Å—Ç—å —Ñ—É–Ω–∫—Ü–∏—è-"–æ–±–µ—Ä—Ç–∫–∞", –∏–≥—Ä–∞–µ—Ç –∑–¥–µ—Å—å –≤–∞–∂–Ω—É—é —Ä–æ–ª—å. –û–Ω–∞ —Ä–µ–∞–ª–∏–∑—É–µ—Ç—Å—è –¥–ª—è –æ–±–µ—Å–ø–µ—á–µ–Ω–∏—è –∏–Ω–∫–∞–ø—Å—É–ª—è—Ü–∏–∏ –∏, —Ç–∞–∫–∏–º –æ–±—Ä–∞–∑–æ–º, —Å–∫—Ä—ã–≤–∞–µ—Ç —Å–µ–±—è –æ—Ç –≥–ª–æ–±–∞–ª—å–Ω–æ–π –æ–±–ª–∞—Å—Ç–∏ –≤–∏–¥–∏–º–æ—Å—Ç–∏.

```python
# decorator function to capitalize names
def names_decorator(function):
   def wrapper(arg1, arg2):
       arg1 = arg1.capitalize()
       arg2 = arg2.capitalize()
       string_hello = function(arg1, arg2)
       return string_hello
   return wrapper

@names_decorator

say_hello('sara', 'ansh')   # output => 'Hello Sara! Hello Ansh!'
```

```python
# output => {11: 121, 3: 9 , 5: 25 , 7: 49}
```

### 25. –ö–∞–∫ —Å–∫–æ–ø–∏—Ä–æ–≤–∞—Ç—å –æ–±—ä–µ–∫—Ç –≤ Python?

–í Python –æ–ø–µ—Ä–∞—Ç–æ—Ä –ø—Ä–∏—Å–≤–∞–∏–≤–∞–Ω–∏—è (`=`) –Ω–µ –∫–æ–ø–∏—Ä—É–µ—Ç –æ–±—ä–µ–∫—Ç—ã. –í–º–µ—Å—Ç–æ —ç—Ç–æ–≥–æ –æ–Ω —Å–æ–∑–¥–∞–µ—Ç —Å–≤—è–∑—å(—Å—Å—ã–ª–∫—É) –º–µ–∂–¥—É —Å—É—â–µ—Å—Ç–≤—É—é—â–∏–º –æ–±—ä–µ–∫—Ç–æ–º –∏ –∏–º–µ–Ω–µ–º —Ü–µ–ª–µ–≤–æ–π –ø–µ—Ä–µ–º–µ–Ω–Ω–æ–π. –ß—Ç–æ–±—ã —Å–æ–∑–¥–∞—Ç—å –∫–æ–ø–∏–∏ –æ–±—ä–µ–∫—Ç–∞ –≤ Python, –Ω–µ–æ–±—Ö–æ–¥–∏–º–æ –∏—Å–ø–æ–ª—å–∑–æ–≤–∞—Ç—å –º–æ–¥—É–ª—å `copy`. –°—É—â–µ—Å—Ç–≤—É–µ—Ç –¥–≤–∞ —Å–ø–æ—Å–æ–±–∞ —Å–æ–∑–¥–∞–Ω–∏—è –∫–æ–ø–∏–π –¥–ª—è –¥–∞–Ω–Ω–æ–≥–æ –æ–±—ä–µ–∫—Ç–∞ —Å –ø–æ–º–æ—â—å—é –º–æ–¥—É–ª—è `copy`.

**Shallow Copy** - —ç—Ç–æ –ø–æ–±–∏—Ç–æ–≤–∞—è –∫–æ–ø–∏—è –æ–±—ä–µ–∫—Ç–∞. –°–æ–∑–¥–∞–Ω–Ω—ã–π —Å–∫–æ–ø–∏—Ä–æ–≤–∞–Ω–Ω—ã–π –æ–±—ä–µ–∫—Ç –∏–º–µ–µ—Ç —Ç–æ—á–Ω—É—é –∫–æ–ø–∏—é –∑–Ω–∞—á–µ–Ω–∏–π –≤ –∏—Å—Ö–æ–¥–Ω–æ–º –æ–±—ä–µ–∫—Ç–µ. –ï—Å–ª–∏ –æ–¥–Ω–æ –∏–∑ –∑–Ω–∞—á–µ–Ω–∏–π —è–≤–ª—è–µ—Ç—Å—è —Å—Å—ã–ª–∫–æ–π –Ω–∞ –¥—Ä—É–≥–∏–µ –æ–±—ä–µ–∫—Ç—ã, –∫–æ–ø–∏—Ä—É—é—Ç—Å—è —Ç–æ–ª—å–∫–æ –∞–¥—Ä–µ—Å–∞ —Å—Å—ã–ª–æ–∫ –Ω–∞ –Ω–∏—Ö.

**–ì–ª—É–±–æ–∫–æ–µ –∫–æ–ø–∏—Ä–æ–≤–∞–Ω–∏–µ** —Ä–µ–∫—É—Ä—Å–∏–≤–Ω–æ –∫–æ–ø–∏—Ä—É–µ—Ç –≤—Å–µ –∑–Ω–∞—á–µ–Ω–∏—è –æ—Ç –∏—Å—Ö–æ–¥–Ω–æ–≥–æ –æ–±—ä–µ–∫—Ç–∞ –∫ —Ü–µ–ª–µ–≤–æ–º—É, —Ç.–µ. –¥—É–±–ª–∏—Ä—É–µ—Ç –¥–∞–∂–µ –æ–±—ä–µ–∫—Ç—ã, –Ω–∞ –∫–æ—Ç–æ—Ä—ã–µ —Å—Å—ã–ª–∞–µ—Ç—Å—è –∏—Å—Ö–æ–¥–Ω—ã–π –æ–±—ä–µ–∫—Ç.

```python
from copy import copy, deepcopy
list_1 = [1, 2, [3, 5], 4]
## shallow copy
list_2 = copy(list_1) 
list_2[3] = 7
list_2[2].append(6)
list_2    # output => [1, 2, [3, 5, 6], 7]
list_1    # output => [1, 2, [3, 5, 6], 4]
## deep copy
list_3 = deepcopy(list_1)
list_3[3] = 8
list_3[2].append(7)
list_3    # output => [1, 2, [3, 5, 6, 7], 8]
list_1    # output => [1, 2, [3, 5, 6], 4]
```

### 26. –í —á–µ–º —Ä–∞–∑–Ω–∏—Ü–∞ –º–µ–∂–¥—É xrange –∏ range –≤ Python?

–í –æ—Ç–ª–∏—á–∏–µ –æ—Ç `range()`, `xrange()` –Ω–µ –≥–µ–Ω–µ—Ä–∏—Ä—É–µ—Ç —Å—Ç–∞—Ç–∏—á–µ—Å–∫–∏–π —Å–ø–∏—Å–æ–∫, –∞ —Å–æ–∑–¥–∞–µ—Ç –∑–Ω–∞—á–µ–Ω–∏–µ –Ω–∞ —Ö–æ–¥—É. –≠—Ç–∞ —Ç–µ—Ö–Ω–∏–∫–∞ –æ–±—ã—á–Ω–æ –∏—Å–ø–æ–ª—å–∑—É–µ—Ç—Å—è —Å –≥–µ–Ω–µ—Ä–∞—Ç–æ—Ä–æ–º –æ–±—ä–µ–∫—Ç–Ω–æ–≥–æ —Ç–∏–ø–∞ –∏ –Ω–∞–∑—ã–≤–∞–µ—Ç—Å—è "**yielding**".


### 28. –ß—Ç–æ —Ç–∞–∫–æ–µ –≥–µ–Ω–µ—Ä–∞—Ç–æ—Ä—ã –≤ Python?

–ü–æ—Å—Ç—Ä–æ–∏–º –≥–µ–Ω–µ—Ä–∞—Ç–æ—Ä –¥–ª—è —á–∏—Å–µ–ª –§–∏–±–æ–Ω–∞—á—á–∏:

```python
## generate fibonacci numbers upto n
def fib(n):
    p, q = 0, 1
    while(p < n):
        yield p
        p, q = q, p + q

x = fib(10)    # create generator object 
 
## iterating using __next__(), for Python2, use next()
x.__next__()    # output => 0
x.__next__()    # output => 1
x.__next__()    # output => 1
x.__next__()    # output => 2
x.__next__()    # output => 3
x.__next__()    # output => 5
x.__next__()    # output => 8
x.__next__()    # error
 
## iterating using loop
for i in fib(10):
    print(i)    # output => 0 1 1 2 3 5 8
```

### 41. –ö–∞–∫ –ø–æ–ª—É—á–∏—Ç—å –¥–æ—Å—Ç—É–ø –∫ —á–ª–µ–Ω–∞–º —Ä–æ–¥–∏—Ç–µ–ª—å—Å–∫–æ–≥–æ –∫–ª–∞—Å—Å–∞ –≤ –¥–æ—á–µ—Ä–Ω–µ–º –∫–ª–∞—Å—Å–µ?

–ù–∏–∂–µ –ø–µ—Ä–µ—á–∏—Å–ª–µ–Ω—ã —Å–ø–æ—Å–æ–±—ã, —Å –ø–æ–º–æ—â—å—é –∫–æ—Ç–æ—Ä—ã—Ö –≤—ã –º–æ–∂–µ—Ç–µ –ø–æ–ª—É—á–∏—Ç—å –¥–æ—Å—Ç—É–ø –∫ —á–ª–µ–Ω–∞–º —Ä–æ–¥–∏—Ç–µ–ª—å—Å–∫–æ–≥–æ –∫–ª–∞—Å—Å–∞ –≤ –¥–æ—á–µ—Ä–Ω–µ–º –∫–ª–∞—Å—Å–µ:

–° –ø–æ–º–æ—â—å—é **–∏–º–µ–Ω–∏ —Ä–æ–¥–∏—Ç–µ–ª—å—Å–∫–æ–≥–æ –∫–ª–∞—Å—Å–∞**: –í—ã –º–æ–∂–µ—Ç–µ –∏—Å–ø–æ–ª—å–∑–æ–≤–∞—Ç—å –∏–º—è —Ä–æ–¥–∏—Ç–µ–ª—å—Å–∫–æ–≥–æ –∫–ª–∞—Å—Å–∞ –¥–ª—è –¥–æ—Å—Ç—É–ø–∞ –∫ –∞—Ç—Ä–∏–±—É—Ç–∞–º, –∫–∞–∫ –ø–æ–∫–∞–∑–∞–Ω–æ –≤ –ø—Ä–∏–º–µ—Ä–µ –Ω–∏–∂–µ:

```python
class Parent(object):
```

–° –ø–æ–º–æ—â—å—é –º–µ—Ç–æ–¥–∞ `super()`: –ß–ª–µ–Ω—ã —Ä–æ–¥–∏—Ç–µ–ª—å—Å–∫–æ–≥–æ –∫–ª–∞—Å—Å–∞ –º–æ–≥—É—Ç –±—ã—Ç—å –¥–æ—Å—Ç—É–ø–Ω—ã –≤ –¥–æ—á–µ—Ä–Ω–µ–º –∫–ª–∞—Å—Å–µ —Å –ø–æ–º–æ—â—å—é –∫–ª—é—á–µ–≤–æ–≥–æ —Å–ª–æ–≤–∞ `super`.

```python
class Parent(object):
   # Constructor
   def __init__(self, name):
       self.name = name    

class Child(Parent):
   # Constructor
   def __init__(self, name, age):         
       ''' 
       In Python 3.x, we can also use super().__init__(name)
       ''' 
       super(Child, self).__init__(name)
       self.age = age
 
   def display(self):
      # Note that Parent.name cant be used 
      # here since super() is used in the constructor
      print(self.name, self.age)

# Driver Code
obj = Child("Interviewbit", 6)
obj.display()
```

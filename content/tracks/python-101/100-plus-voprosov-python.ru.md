### 22. –ß—Ç–æ —Ç–∞–∫–æ–µ –¥–µ–∫–æ—Ä–∞—Ç–æ—Ä—ã –≤ Python?

–î–µ–∫–æ—Ä–∞—Ç–æ—Ä—ã –≤ Python - —ç—Ç–æ, –ø–æ —Å—É—Ç–∏, —Ñ—É–Ω–∫—Ü–∏–∏, –∫–æ—Ç–æ—Ä—ã–µ –¥–æ–±–∞–≤–ª—è—é—Ç —Ñ—É–Ω–∫—Ü–∏–æ–Ω–∞–ª—å–Ω–æ—Å—Ç—å –∫ —Å—É—â–µ—Å—Ç–≤—É—é—â–µ–π —Ñ—É–Ω–∫—Ü–∏–∏ –≤ Python –±–µ–∑ –∏–∑–º–µ–Ω–µ–Ω–∏—è —Å—Ç—Ä—É–∫—Ç—É—Ä—ã —Å–∞–º–æ–π —Ñ—É–Ω–∫—Ü–∏–∏. –í Python –æ–Ω–∏ –æ–±–æ–∑–Ω–∞—á–∞—é—Ç—Å—è @decorator_name –∏ –≤—ã–∑—ã–≤–∞—é—Ç—Å—è –ø–æ –ø—Ä–∏–Ω—Ü–∏–ø—É "—Å–Ω–∏–∑—É –≤–≤–µ—Ä—Ö". –ù–∞–ø—Ä–∏–º–µ—Ä:

```python
# decorator function to convert to lowercase
def lowercase_decorator(function):
   def wrapper():
       func = function()
       string_lowercase = func.lower()

       func = function()

   return 'Hello World'

```

–ü—Ä–µ–ª–µ—Å—Ç—å –¥–µ–∫–æ—Ä–∞—Ç–æ—Ä–æ–≤ –∑–∞–∫–ª—é—á–∞–µ—Ç—Å—è –≤ —Ç–æ–º, —á—Ç–æ –ø–æ–º–∏–º–æ –¥–æ–±–∞–≤–ª–µ–Ω–∏—è —Ñ—É–Ω–∫—Ü–∏–æ–Ω–∞–ª—å–Ω–æ—Å—Ç–∏ –∫ –≤—ã—Ö–æ–¥—É –º–µ—Ç–æ–¥–∞, –æ–Ω–∏ –º–æ–≥—É—Ç –¥–∞–∂–µ –ø—Ä–∏–Ω–∏–º–∞—Ç—å –∞—Ä–≥—É–º–µ–Ω—Ç—ã –¥–ª—è —Ñ—É–Ω–∫—Ü–∏–π –∏ –¥–æ–ø–æ–ª–Ω–∏—Ç–µ–ª—å–Ω–æ –º–æ–¥–∏—Ñ–∏—Ü–∏—Ä–æ–≤–∞—Ç—å —ç—Ç–∏ –∞—Ä–≥—É–º–µ–Ω—Ç—ã –ø–µ—Ä–µ–¥ –ø–µ—Ä–µ–¥–∞—á–µ–π –≤ —Å–∞–º—É —Ñ—É–Ω–∫—Ü–∏—é. –í–Ω—É—Ç—Ä–µ–Ω–Ω—è—è –≤–ª–æ–∂–µ–Ω–Ω–∞—è —Ñ—É–Ω–∫—Ü–∏—è, —Ç–æ –µ—Å—Ç—å —Ñ—É–Ω–∫—Ü–∏—è-"–æ–±–µ—Ä—Ç–∫–∞", –∏–≥—Ä–∞–µ—Ç –∑–¥–µ—Å—å –≤–∞–∂–Ω—É—é —Ä–æ–ª—å. –û–Ω–∞ —Ä–µ–∞–ª–∏–∑—É–µ—Ç—Å—è –¥–ª—è –æ–±–µ—Å–ø–µ—á–µ–Ω–∏—è –∏–Ω–∫–∞–ø—Å—É–ª—è—Ü–∏–∏ –∏, —Ç–∞–∫–∏–º –æ–±—Ä–∞–∑–æ–º, —Å–∫—Ä—ã–≤–∞–µ—Ç —Å–µ–±—è –æ—Ç –≥–ª–æ–±–∞–ª—å–Ω–æ–π –æ–±–ª–∞—Å—Ç–∏ –≤–∏–¥–∏–º–æ—Å—Ç–∏.

```python
# decorator function to capitalize names
def names_decorator(function):
   def wrapper(arg1, arg2):
       arg1 = arg1.capitalize()
       arg2 = arg2.capitalize()
       string_hello = function(arg1, arg2)
       return string_hello
   return wrapper

say_hello('sara', 'ansh')   # output => 'Hello Sara! Hello Ansh!'
```

–°–ª–æ–≤–∞—Ä–Ω—ã–µ (Dict) comprehension –∏—Å–ø–æ–ª—å–∑—É—é—Ç —Ñ–∏–≥—É—Ä–Ω—ã–µ —Å–∫–æ–±–∫–∏ –∏ –ø–æ–∑–≤–æ–ª—è—é—Ç —Å–æ–∑–¥–∞–≤–∞—Ç—å –Ω–æ–≤—ã–µ —Å–ª–æ–≤–∞—Ä–∏ –Ω–∞ –æ—Å–Ω–æ–≤–µ —É–∂–µ —Å—É—â–µ—Å—Ç–≤—É—é—â–∏—Ö.

```python
new_dict = {key: value for key, value in old_dict.items() if value > 2}
```

–í—ã–ø–æ–ª–Ω–µ–Ω–∏–µ –º–∞—Ç–µ–º–∞—Ç–∏—á–µ—Å–∫–∏—Ö –æ–ø–µ—Ä–∞—Ü–∏–π –Ω–∞–¥ –≤—Å–µ–º —Å–ø–∏—Å–∫–æ–º

```python
my_list = [2, 3, 5, 7, 11]
squared_list = [x**2 for x in my_list]    # list comprehension
# output => [4 , 9 , 25 , 49 , 121]
squared_dict = {x:x**2 for x in my_list}    # dict comprehension
# output => {11: 121, 2: 4 , 3: 9 , 5: 25 , 7: 49}
```

–í—ã–ø–æ–ª–Ω–µ–Ω–∏–µ –æ–ø–µ—Ä–∞—Ü–∏–π —É—Å–ª–æ–≤–Ω–æ–π —Ñ–∏–ª—å—Ç—Ä–∞—Ü–∏–∏ –¥–ª—è –≤—Å–µ–≥–æ —Å–ø–∏—Å–∫–∞

```python
my_list = [2, 3, 5, 7, 11]
squared_list = [x**2 for x in my_list if x%2 != 0]    # list comprehension
# output => [9 , 25 , 49 , 121]
squared_dict = {x:x**2 for x in my_list if x%2 != 0}    # dict comprehension
# output => {11: 121, 3: 9 , 5: 25 , 7: 49}
```
